## ğŸš€ Deployment to GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and create a new repository
2. Name it `portfolio` (or any name you prefer)
3. Make it public

### Step 2: Clone and Add Files
```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
```

### Step 3: Copy Portfolio Files
Copy the three files (index.html, styles.css, script.js) to your repository folder

### Step 4: Push to GitHub
```bash
git add .
git commit -m "Initial portfolio commit"
git push origin main
```

### Step 5: Enable GitHub Pages
1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Your portfolio will be live at: `https://YOUR_USERNAME.github.io/portfolio/`

## ğŸ¨ Customization

### Update Personal Information
Edit `index.html` and replace:
- Your name and title
- About me text
- Experience details
- Project information
- Contact information
- Social media links

### Change Colors
Edit `:root` variables in `styles.css`:
```css
:root {
    --primary: #6366f1;        /* Main color */
    --secondary: #ec4899;      /* Accent color */
    --accent: #f59e0b;         /* Highlight color */
    /* ... more colors ... */
}
```

### Add Your Photo
Replace the placeholder image URL in the About section:
```html
<img src="YOUR_IMAGE_URL" alt="Your Name">
```

### Update Project Links
Replace `#` with actual GitHub and demo links in project cards
